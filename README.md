# e2e-test-cardX

Cypress tests for demo (UI and API)

## Requirements
- Node.js >= 18
- Git

## Setup
```bash
git clone https://github.com/ganixtanzard-max/e2e-test-cardX.git
cd e2e-test-cardX
npm ci
npm run verify

Run Tests
Interactive (เปิด Cypress GUI):
npm run cy:open
Headless (CI mode):
npm run cy:run
